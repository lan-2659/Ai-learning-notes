

爬虫：
模拟浏览器，发送请求，获取响应
只能获取网页展示出来的内容，网页隐藏的内容需要用其它手段获取

爬虫分类：
    通用爬虫：爬取网站无上限
    聚焦爬虫：爬取网站数量有上限，有明确的目标
        功能性爬虫：不获取数据，只为实现某一功能
        数据增量爬虫：获取数据用于后续分析

写爬虫程序步骤：
1、获取网页内容
HTTP请求，Requests库
2、解析网页内容
HTML网页结构，BeautifulSoup库
3、对得到的内容进行其它操作

爬虫使用规则：
1、不要爬取公民隐私数据
2、不要爬取受著作权保护的内容
3、不要爬取国家事务、国防建设、尖端技术领域的计算机系统等
4、自己写的爬虫的请求频率和数量不能过高（过高就类似于DDos攻击）
5、不要去突破网站的反爬机制（如验证码、登录等机制）


# 可以通过查看网页的robots.txt文件(君子协议)了解可爬取的网页路径范围

# HTTP：超文本传输协议（Hypertext Transfer Protocol），这是一种客户端与服务器之间的请求-响应协议
# HTTPS（HTTP + SSL）：带安全套接字层的超文本传输协议
# HTTPS 比 HTTP 更安全，但性能更低
# HTML: 定义网页的结构和信息
